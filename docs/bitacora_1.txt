La idea es generar un agente de IA con la herramienta de desarrollo de Notion que permite conectar desarrollos a su platafoma.
La arquitectura de este agente esta dividido en tres capas:
    1. Frontend: interfaz web simple en HTML/CSS/JS.
    2. Backend: Python con el modelo IA como cerebro. Modelo decide qué herramienta usar según
        lo que el usuario pide. Las herramientas son funciones Python que llaman a la API de Notion.
    3. Notion API: integración real contra el workspace usando el SDK oficial notion-client.
El flujo que se utilizará es: usuario escribe -> modelo IA interpreta -> el modelo llama a la herramienta ->
                                la función se ejecuta en Notion -> el usuario recibe respuesta del modelo al chat

Para arrancar con el proyecto comenzamos creando una cuenta de Notion limpia y configurar una "Internal Integration" dentro de
la plataforma de notion, la cual nos servirá para obtener una llave de autorización para poder ejecutar funciones dentro de las
paginas de notion. (https://www.notion.so/my-integrations)

    Nombre de la integración: Agente IA (Demo 1)
    Associated workspace: Juan Pablo's Notion
    Capacidades: CRUD completo dentro de las páginas autorizadas

Una vez que creamos la integración se tuvo que generar "datos prueba" o una "base de conocimiento". Para ello creamos una página
principal y otras 3 subpáginas donde guardamos 3 tipos de datos; reuniones, ideas del proyecto y tareas pendientes. Una vez generado
lo anterior se configuró la pagina principal para poder ser conectada con la integracion anteriormente creada.

Una vez configurado Notion necesitamos obtener otra llave de acceso para el modelo de IA que usaremos, por el momento la UVM nos
brinda con modelos potentes de Google Gemini gracias a un convenio con la institucion y la empresa tecnologica. Para ello necesitamos
obtenerla desde la pagina de google studio iniciando sesion con el correo institucional y solicitar la API Key (https://aistudio.google.com/)

    Nombre de la API Key: agente-notion
    Proyecto importado: agente-notion-demo

Después de tener todo esto listo debemos de comenzar a crear el proyecto Python, para ello hicimos los siguientes pasos:
    1. Abrir terminal en la ubicacion donde queremos que se encuentre el proyecto
    2. Ejecutar los siguientes comandos:
        mkdir agente-notion                                                     (Crear la carpeta del proyecto)
        cd agente-notion                                                        (Moverse a la carpeta del proyecto)
        python -m venv venv                                                     (Aislar dependencias de python para no chocar con otras instalaciones de python)
        venv\Scripts\activate                                                   (Enciende el entorno virtual)
        pip install notion-client google-generativeai streamlit python-dotenv   (Instalar dependencias)
    3. Dentro de la carpeta del proyecto crearemos un archivo de variables de entorno con las API Keys que generamos de notion y gemini

Una vez establecida la configuracion del entrono y de accesos debemos definir la estructura del proyecto:
    agente-notion/
    ├── venv/
    ├── .env
    ├── notion_tools.py      ← las 3 funciones que tocan Notion
    ├── agent.py             ← el cerebro: Gemini + tool use
    └── app.py               ← la interfaz Streamlit
